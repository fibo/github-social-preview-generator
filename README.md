# GitHub Social Preview generator

> Create a GitHub Social Preview image, via HTML/CSS and Chrome "Capture node screenshot" feature

## Instructions

1. With Chrome, go to https://fibo.github.io/github-social-preview-generator
2. Select image (.e.g. npm, shell, etc.)
3. Edit project name and description
4. Right click the generated preview to open _DevTools_ then right click the Element node and select __Capture node screenshot__
5. Finally go to your repository _Settings > General > Social preview_ and upload the image
